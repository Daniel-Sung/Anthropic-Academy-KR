- YouTube iframe `loading="lazy"` 적용
- 추천 학습 순서: `navigateTo(tabId, sectionId)` 함수로 탭 전환 + 섹션 열기 + 스크롤 + 하이라이트

- 설명 보강은 Python 스크립트(`enhance_desc.py`)로 일괄 처리 후 삭제
