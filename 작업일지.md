### 기술 메모 (Day 2 추가)
- Firebase SDK: compat v10.14.1 (script 태그, MyRoutine과 동일 패턴)
- 데이터 구조: `users/{uid}` → `{ progress, memos, lastSync, displayName, email }`
- 메모 키: `anthropic-academy-memos` (localStorage)
- 동기화: `debouncedSync()` 3초 디바운스 → `syncToCloud()`
- 첫 동기화: 로컬/클라우드 양쪽 데이터 있으면 사용자에게 선택 요청
- 메모 UI: JS로 동적 생성 (`initMemos()`), `.learn-item-wrap` > `.learn-item-top` + `.memo-area` 구조

### 기술 메모 (Day 1)
- Pretendard Variable 폰트 사용 (CDN)
- 색상: Anthropic 브랜드 오렌지(#D97757) 기반
